# Multi-Gateway-Payment-Engine
Multi-Gateway Payment Orchestration Engine

A scalable backend system that orchestrates payment processing across multiple gateways with intelligent routing, retry mechanisms, and state management.

🚀 Features
🔀 Multi-Gateway Routing
Dynamically routes transactions based on configurable rules
🔁 Adaptive Retry & Failover
Retries failed transactions with fallback gateway selection
🔄 Transaction State Management
Tracks lifecycle: INIT → PROCESSING → SUCCESS / FAILED
🧱 Modular Architecture
Clean separation of orchestration, routing, and gateway layers
🛡️ Fault Tolerance
Handles failures and ensures reliable transaction processing
